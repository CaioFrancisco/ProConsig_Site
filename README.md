## 🚀 About the Project  

ProConsig is a modern, conversion-driven web application designed to simplify the loan simulation process and instantly connect users with a consultant via WhatsApp.  

Instead of complex systems and long forms, the platform focuses on **speed, simplicity, and human interaction**.  

> The goal is simple: reduce friction and increase conversion.

---

## ✨ Key Features  

- 📊 Smart loan simulation form  
- 📱 WhatsApp redirection with pre-filled message  
- ⚡ Fast user journey (no login required)  
- 🎯 Conversion-focused design  
- 🎨 Clean and modern UI (green & white theme)  
- 🔔 Informational popup ("Why choose us")  
- 📌 Transparent legal disclaimer
